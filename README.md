# ImageWatermark

一个面向 Windows 的轻量级图片水印工具，基于 `PySide6 + Pillow` 实现。  
它支持给常见图片快速添加自定义文字水印，并提供实时预览与批量导出能力，适合个人创作、素材保护和团队内部图片流转场景。

## 1. 工具简介

`ImageWatermark` 是一个桌面端水印应用，当前仅适配 Windows。  
你可以在图形界面中完成“选择图片 -> 配置水印 -> 导出结果”的完整流程，无需命令行参与日常使用。

该项目同时提供了可执行文件打包与安装脚本，便于后续分发给非开发人员使用。

## 2. 工具功能列表

- 支持主流图片格式：`jpg`、`jpeg`、`png`、`bmp`、`webp`
- 支持单张图片加水印并导出
- 支持批量目录处理并统一导出
- 水印文本可自定义
- 水印样式可调：
  - 字体
  - 字号
  - 颜色（预设常见颜色）
  - 透明度
  - 旋转角度
- 支持两种布局模式：
  - 九宫格定位（如右下角、居中等）
  - 平铺模式（可调水平/垂直间距）
- 支持水印密度调节（平铺模式）：
  - 密度高：水印更密集
  - 密度低：水印更稀疏
- 支持实时预览，并可切换查看“原图/水印图”
- 支持内置字体加载：
  - 从 `assets/fonts` 目录加载自带 `.ttf` 字体
  - 未提供字体文件时，自动回退到系统字体（含中文友好回退，如 `Microsoft YaHei`、`SimHei`、`SimSun`）
- 提供 Windows 打包与安装能力：
  - `PyInstaller` 生成可执行程序
  - `Inno Setup` 生成安装包

## 3. 如何上手使用

### 3.1 环境准备

- 操作系统：Windows
- Python：建议 `3.10+`

### 3.2 安装依赖

```bash
python -m pip install -r requirements.txt
```

### 3.3 启动工具

```bash
python app/main.py
```

### 3.4 基本使用流程

1. 点击“选择单张图片”或“选择批量目录”
2. 在右侧面板设置水印文本、字体、字号、颜色、透明度、密度等参数
3. 在中间预览区域确认效果（可切换原图/水印图）
4. 点击“导出单张”或“批量导出”完成输出

### 3.5 打包为可执行程序和安装包

1. 执行构建脚本：

```powershell
.\build.ps1
```

2. 使用 Inno Setup 打开并编译：

- `installer/InnoSetup.iss`

编译完成后即可得到 Windows 安装程序。
